JavaScript Events (onTouchStart) and Animation and Transition Events

Events

JavaScript can be executed when an event occurs. Some examples of HTML events that can trigger JavaScript are:

  • Mouse click
  • Mouse over an element
  • Web page has loaded
  • Input field changed
  • Form submitted
  • Key pressed on the keyboard
  • Mobile screen pressed
  • etc...

To execute code when an event occurs an HTML event attribute can be added to the element. For example, if we want JavaScript to execute when an element is clicked we might do this:

<button onclick="this.innerHTML = 'Ouch!'">Don't Click Me</button>

Alternatively, we might add an event listener to the element to execute the JavaScript. This is done using the addEventListener() method.

document.getElementById("myButton").addEventListener("click", ouch);
<button id="myButton">Don't Click Me</button>
