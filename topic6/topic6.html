DOM Manipulation Using createElement, appendChild, insertBefore etc.

What is DOM Manipulation?

The Document Object Model defines a standard for accessing elements in a document. HTML DOM can facilitate getting, changing, adding, or deleting HTML elements. Used in conjuction with JavaScript it is possible to create dynamic HTML. It can:

  • Change all the HTML elements in the page
  • Change all the HTML attributes in the page
  • Change all the CSS styles in the page
  • Remove existing HTML elements and attributes
  • Add new HTML elements and attributes
  • React to all existing HTML events in the page
  • Create new HTML events in the page
When a page is loaded the browser converts the HTML into a tree structure. For example, the following HTML would be represented as the tree below.

<html>
<head>
<title>CIT 261 is Great!</title>
</head>
<body>
<h1>Learning DOM</h1>
<p>Manipulate everything.</p>
</body>
</html>


The nodes in this tree have a hierarchical relationship. These relationships are described as parent, child, and sibling. In this tree <html> is the root element and has no parent. The <html> node has two children, <head> and <body>. Those two nodes are also siblings and are element nodes. The leaf nodes in this diagram are considered text nodes

createElement(), appendChild(), insertBefore()

The createElement() method is used to create a new element node For example, to create a new <p> element:

var paragraph = document.createElement("P");

After a new element is created it can be inserted into the document using the appendChild() or insertBefore() methods. The insertBefore() method will insert a node as a child right before the specified node. The appendChild() method will insert a node as the last child of the specified node.

Code Example:
We have a list that contains a list of sports, and we want to add a new element to that list.

<ul id="list">
<li>Football</li>
<li>Hockey</li>
<li>Basketball</li>
<li>Soccer</li>
</ul>

First, create a new element.

var li = document.createElement("LI");

Next a text node needs to be created and appended to the newly created <li> node.

var text = document.createTextNode("Baseball");

li.appendChild(text);

Now the <li> node can be appended to the <ul> node with the specified id.

document.getElementById("list").appendChild(li);



The resulting list is below.

  • Football
  • Hockey
  • Basketball
  • Soccer
