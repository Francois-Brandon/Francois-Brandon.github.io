For Loop

There are 3 main components to a for loop.

  1. Loop Inititialization
  2. Test Statement
  3. Iteration Statement
for (initialization; test condition; iteration statement){
Code to be executed;
}

The loop initialization sets the starting value for counter.
The test statement defines the condition to either execute the code inside the loop or exit the loop.
The iteration statement increments or decrements the counter value.

For/In Loop

The for...in loop is used to loop throught the properties of a JavaScript Object.
for (variable in object){
code to be executed;
}

For each loop a property will be assigned to the variable. The code inside the loop will be executed until it has gone through every property.
