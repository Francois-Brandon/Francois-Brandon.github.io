For Loop

There are 3 main components to a for loop.

  1. Loop Inititialization
  2. Test Statement
  3. Iteration Statement
Syntax:
for (initialization; test condition; iteration statement){
Code to be executed;
}

The loop initialization sets the starting value for counter.
The test statement defines the condition to either execute the code inside the loop or exit the loop.
The iteration statement increments or decrements the counter value.

Code Example:
Press the button to execute a counting for loop.
The loops starts with the counter at 0 and will run as long as it is less than 5.

For/In Loop

The for...in loop is used to loop throught the properties of a JavaScript Object.

Syntax:
for (variable in object){
code to be executed;
}

For each loop a property will be assigned to the variable. The code inside the loop will be executed until it has gone through every property.

Code Example:
Press the button to loop through all the properties for the player object.

While Loop

The while loop will execute the statements or code block as long as the expression evaluates as true.

Syntax:
while (expression){
code to be executed;
}

Code Example:
A variable i is set to 0. The while loop will run while i is less than 5. The variable i will be incremented each time the loop is run. Press the button to see the output.

Do/While Loop

Like the while loop, the do/while loop will execute the statements or code block as long as the expression evaluates as true. The difference is that the expression is evaluated after the loop is run. This will guarantee that the code or statements in the loop will be executed at least once

Syntax:
do{
Statement(s) to be executed;
} while (expression);

Code Example:
A variable i is set to 5. The do/while loop will run then evaluate if i is less than 5. Since this will evaluate to false the loop will be exited. Press the button to see the output.
