For Loop

There are 3 main components to a for loop.

  1. Loop Inititialization
  2. Test Statement
  3. Iteration Statement
Syntax:
for (initialization; test condition; iteration statement){
Code to be executed;
}

The loop initialization sets the starting value for counter.
The test statement defines the condition to either execute the code inside the loop or exit the loop.
The iteration statement increments or decrements the counter value.

Code Example:
Press the button to execute a counting for loop.
The loops starts with the counter at 0 and will run as long as it is less than 5.

For/In Loop

The for...in loop is used to loop throught the properties of a JavaScript Object.

Syntax:
for (variable in object){
code to be executed;
}

For each loop a property will be assigned to the variable. The code inside the loop will be executed until it has gone through every property.

Code Example:
Press the button to loop through all the properties for the player object.

While Loop

The while loop will execute the statements or code block as long as the expression evaluates as true.

Syntax:
while (expression){
code to be executed;
}

Code Example:
A variable i is set to 0. The while loop will run while i is less than 5. The variable i will be incremented each time the loop is run. Press the button to see the output.

Do/While Loop

Like the while loop, the do/while loop will execute the statements or code block as long as the expression evaluates as true. The difference is that the expression is evaluated after the loop is run. This will guarantee that the code or statements in the loop will be executed at least once

Syntax:
do{
Statement(s) to be executed;
} while (expression);

Code Example:
A variable i is set to 5. The do/while loop will run then evaluate if i is less than 5. Since this will evaluate to false the loop will be exited. Press the button to see the output.

If, Else, Else If

The if statement is used to execute a block of code if a condition evaluates to true.

Syntax:
if (condition) {
block of code to be executed if the condition is true
}

 The else statement is used in conjunction with the if statement. The else statement is placed after the if statement. The code in the else statement would be executed as an alternative to the code in the if statement, that is, if the condition evaluates to false

Syntax:
if (condition) {
code to be executed if the condition is true
} else {
code to be executed if the condition is false
}

 The else if statement is used to declare additional conditions after the first. If the first condition is false the next condition will be checked and so on until a true condition is found, else statement is reached, or all conditions are evaluated as false.

Syntax:
if (condition1) { block of code to be executed if condition1 is true } else if (condition2) { code to be executed if the condition1 is false and condition2 is true } else { code to be executed if the condition1 is false and condition2 is false }

Switch

As an alternative to a series of if...else if statements a switch statement can be used. As you can see in the syntax below, the switch statement will take the value of the expression and match it with the cases. Each case will have a statement or code to execute if its condition is met. After each case statement the keyword break is used. This signifies the end of that particular case. Without the break statements each case following the true condition would be executed.

Syntax:
switch (expression)
{
case condition 1: statement(s)
break;

case condition 2: statement(s)
break;
...

case condition n: statement(s)
break;

default: statement(s)
}

Functions

Functions can be used to group code into reusable blocks. A function is defined with the function keyword followed by the name of the function. Immediately following the function name are parenthesis where any parameters being sent to the function are defined. The body or code block of the function is surrounded by curly braces. There is also the optional keyword return. This would allow a value to be returned from the function.

Syntax:
function name(parameters, are, commaSeparated) {
statements;
return something; //optional
}

Arrays

An array is an object that allows you to store many values in a single variable.

Syntax:
To create an array you use square brackets and separate the different values using commas.
var array_name = [item1, item2, ...];

An array can also be created using the new keyword. If only a single number is used as a parameter then the array will be created with that length.
var array_name = new Array(item1, item2, ...);

When an array is created the values are stored sequentially with the first value occupying the position with index [0]. The various elements can be accessed by referring to the index number. This can be used to either get or set the specified element. To access the second element you would use: array_name[1];

There are many built in array properties and methods. For example, the length preperty will return the number of elements in the array.
var tools = ["hammer", "saw", "screwdriver", "wrench"];
var length = tools.length;
document.write("The length is " + length);


A summary of some other available properties/methods:

  • forEach() - Loop over an Array
  • push() - Add to the end of an Array
  • pop() - Remove from the end of an Array
  • shift() - Remove from the front of an Array
  • unshift() - Add to the front of an Array
  • indexOf() - Find the index of an item in the array
  • concat() - Creates a new array by merging existing arrays
