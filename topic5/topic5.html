Local Storage: Storing and Retrieving Arrays, Associative Arrays, and Objects

Example 1: Using the Web Storage API and JSON to Store and Retrieve Associative Array.

The following example demonstrates storing and retreiving arrays using Local Storage. Under each picture there is a text box to enter a caption. After a caption is entered for each picture press the Save Captions button. The captions will not be serialized, stored in Local Storage, retrieved from storage, deserialized and displayed under the picture. To test that the captions were actually stored and retreived refresh the page and press the Display Saved Captions button.
