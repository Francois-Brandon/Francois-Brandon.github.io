Javascript - Object Creation Functions, Properties, Methods, Instantiation

Object Creation/Instantiation

Along with the predefined objects available in Javascript you can create your own objects. Two ways to create an object are by using an object initializer or a constructor function. Using an object initializer is also known as creating an object with literal notation.

Object Initializer Syntax:

var object = {property1: value1, property2: value2, ...}

Each object initializer results in a new object being created. If the same object initializer is used it will still create a separate and distinct object. Comparing the two will result in a false equality. These objects are created as if new Object() were used.

To create an object using a constructor function you would first need to define the object by creating a constructor function. Next, you can create an instance of the object by using the new keyword.

Constructor Function Syntax:

function Player(name, health, mana, gold) {
this.name = name;
this.health = health;
this.mana = mana;
this.gold = gold;
}

var newPlayer = new Player('Brandon', 10, 20, 50)

Object Properties

In the examples above, not only are objects being created, but properties are being assigned to that object as well. A property can be described as a variable attached to the object. These properties define the object's characteristics. Two ways to access object properties are with the dot and bracket notations.

newPlayer.name //returns 'Brandon'
newPlayer['health'] //returns 10

Object Methods

A method is a property of an object that is a function. They are defined the way as a normal function, but are assigned as an object property. Methods can be defined for an existing object or can be included in the object constructor function.

Existing Object Syntax:

objectName.method = function;

Literal Notation Syntax:

var object = { property: value, method(param) {
// function body }
};

Object Constructor Syntax:

function Player(name, health, mana, gold) {
this.name = name;
this.health = health;
this.mana = mana;
this.gold = gold;
this.sethealth = setHealth;
}

function setHealth(delta) {
this.health += delta;
if (this.health > 30) {
this.health = 30;
}
else if (this.health < 0) {
this.health = 0;
}
