JSON: Parse, Stringify

What is JSON?

JSON stands for JavaScript Object Notation. It is a data-interchange format, meaning it is primarily used to transmit data in a consistent format. JSON is lngauge independent. The JSON format was developed to be easy for humans to read and write as well as to be easy for machines to parse and generate.

There are two main structures to JSON:

  • A collection of name/value pairs, which can be interpreted as:
    • Object
    • Record
    • Struct
    • Dictionary
    • Hashed Table
    • Keyed List
    • Associative Array

  • Ordered list of values, which can be interpreted as:
    • Array
    • Vector
    • List
    • Sequence
Syntax:
  • Object
    • { string:value, string:value, ... }
    • Pairs are comma separated
  • Array
    • [value, value, ...]
    • Values are comma separated
  • Values formats:
    • String
    • Number
    • Object
    • Array
    • True
    • False
    • null
  • String formats:
    • Any number of Unicode characters except " or \ or control character surrounded by double quotes
    • Supports backslash escapes. (i.e. \n,\t, etc.)
  • Number formats:
    • Digits (0-9)
    • Numbers with decimals (.123)
    • Exponents e (1e05)

JSON Parse

JSON Parse is used to convert a string in a JSON format into a JavaScript object. When using JSON.parse() on a JSON that derived from an array a JavaScript array will be returned instead. If the string is not in a JSON format then the parse will fail with a syntax error.

Syntax:
JSON.parse(text[, reviver])

Parameters:

  • Text: the string that will be parsed.
  • Reviver: (optional) When it is defined as a function it can be used to transform a parsed value before it is returned.
Code Examples:
Create a string in a proper format:

var text = '{"name":"Brandon","health":20,"mana":30,"gold":50,"weapon":"staff"}';

Now parse that string into a JavaScript object:

var obj = JSON.parse(text);

We can now loop through all the properties in the object and display them:

var p;
for (p in obj) {
document.write(p + "<br>");
}

JSON Stringify

Stringify is a Javascript function used to convert a JavaScript object into a string. This is useful when sending data to a web server as the data needs to be a string.

Syntax:
JSON.stringify(value[, replacer[, space]])

Parameters:

  • Value: the value that is being converted to a string
  • Replacer:(optional) This can be a function that modifies the stringification process or an array of String or Number Objects. The latter would serve as a whitelist for filtering the properties of value as it is converted to a string. If no replacer is specified then all properties will be included.
  • Space: (optional) A string or number can be input here. If a number is provided this will specify the number of space characters to use as spaces. The number is capped at 10. Anything more than 10 will be capped at 10. If a string is provided that string will be used as white space. Again, it is capped at 10 characters. If more than 10 characters are provided only the first 10 will be used. If no space is specified then no white space woul be used.
String, Boolean, and Number objects are converted to their primitive values during stringification. When an object is being converted and undefined, a function, or a symbol are found these will be omitted. If those conditions are found when passing pure values they will be represented as null.

Code Example:
First, lets create an object.

var player = { name:"Brandon", health:20, mana:30, gold:50, weapon:"staff" };

Now let's stringify the object.

var playerJson = JSON.stringify(player);
